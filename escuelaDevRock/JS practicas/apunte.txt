JavaScript Asíncrono.

JavaScript permite que trabajemos de forma asincrona. 
Esto es: cuando le solicitamos que haga tareas que pueden llevar un tiempo "largo" js continua con otras tareas mientras esta tarea mas larga finaliza.

JS no es multihilo, es asincronico y concurrente.
